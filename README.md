# Pain Management RCT Analysis

# This repository contains R scripts for analyzing a randomized controlled trial (RCT) in pain management.

## Contents
1. `rct_data_cleaning.R` – Data cleaning and summary statistics
2. `rct_analysis.R` – Statistical analysis for RCT outcomes
3. `rct_visualization.R` – Visualization of pain scores over time

## Tools
- R
- dplyr
- tidyr
- ggplot2
- broom
